# Car_Project

This is a simple C project to manage a collection of cars. It allows you to add, modify, delete, display, and search for cars based on their brand.
## Features
- Add a new car with details such as brand, model, year, and mileage.
- Modify existing car details.
- Delete a car from the collection.
- Display all cars in the collection.
- Search for cars by brand (case-insensitive).
## Compilation
To compile the project, use the following command:
```
bashgcc main.c -o car_manager
```
Or for Windows:
```
bashgcc main.c -o car_manager.exe
```
## Usage
Run the compiled program:
```
bash ./car_manager
```
Or for Windows:
```
bash car_manager.exe
```

Follow the on-screen menu to manage your car collection. 

## Note
Make sure to handle memory management properly to avoid memory leaks.
